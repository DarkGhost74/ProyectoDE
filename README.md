## Descripción del proyecto
Este proyecto es una aplicación desarrollada en R Shiny que permite resolver problemas de regresión lineal simple y múltiple de manera interactiva. El usuario puede cargar un conjunto de datos, seleccionar la variable dependiente y las variables independientes, y la aplicación calcula automáticamente el modelo.<br><br>
La interfaz está construida en ui.R, la lógica de cálculo y generación de gráficos en server.R, y app.R sirve como archivo principal para ejecutar la aplicación. El sistema muestra resultados numéricos, tablas interactivas mediante la librería DT, y gráficas generadas con ggplot2, facilitando el análisis visual y estadístico dentro del navegador sin necesidad de programar manualmente en R.

## Librerías necesarias
Para la correcta ejecución de este proyecto es necesario tener instaladas las siguientes librerías:
- shiny
- DT
- ggplot2
